meta Object 문서 정보
 - version String 버전 정보 # 야놀자 전체 목록 가져온 일자로 설정, 일단 YaDeals.creation_time 일자부분으로 사용
 packageId String 패키지 고유 번호, !!GDF가 아닌 신규 필드!! # 'YNJ' + YaDeal.id 값으로 사용 
 prices Array 상품별 가격 정보
 - {productId} Array 상품 고유 번호 # 'YNJPRO' + YaDealOption.id 값으로 사용 
 - - {array-index} Array-Object 가격 개별 리스트 정보 
 - - - id String 가격 고유 번호 # 'YNJPRI' + YaDealOptionItem.id 값으로 사용
 - - - label Object 사용자 정보 
 - - - - en String 사용자 정보(영어) # null로 설정
 - - - - ko String 사용자 정보(한글) # YaDealOptionItem.name
 - - - policy Object 정책 정보 
 - - - - age Object 나이 정보 
 - - - - - min Number 최소 나이 정보 나이 정보를 받지 못했을 경우 값 : -1 # -1로 설정 
 - - - - - max Number 최대 나이 정보 나이 정보를 받지 못했을 경우 값 : -1 # -1로 설정
 - - - priceInfo 판매 가능 날짜 정보 신규 필드
 - - - - from String '2020-10-13 00:00:00' 신규 필드
 - - - - to String '2020-12-13 00:00:00' 신규 필드
 - - - - price Number 공급가격 신규 필드
 - - - - salePrice Number 판매 가격 신규 필드
 - - - - markup Number 판매 가격 - 공급 가격 신규 필드
 - - - - displayMarkup Number 정상가격 - 판매가격 신규 필드
 - - - dates Array 예약 가능 날짜 정보 빈 배열로 처리한다
 - - - - {array-index} Array-Object 날짜별 리스트 정보 더이상 생성하지 않는다
 - - - - - date String 날짜 정보 # "2020-10-13" 형식으로 일자별 처리 더이상 생성하지 않는다
 - - - - - price Number 공급 가격 정보 # DealOptionItem.cost_price 값 더이상 생성하지 않는다
 - - - - - salePrice Number 판매 가격 정보 신규 필드 더이상 생성하지 않는다
 - - - - - markup Number 판매가-공금가격 정보 향후 정산을 위한 신규 필드 더이상 생성하지 않는다
 - - - - - available Boolean 이용 가능 정보 # true 설정 # TODO 정확한 값 필요시 야놀자 문의후 costPrice가 0보다 크고 pqlat, qplpp이 존재하며 0이 아니면 True 로 설정한다.*더이상 생성하지 않는다*
