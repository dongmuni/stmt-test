meta Object 문서 정보
 - version String 데이터 버전 정보 # 야놀자 전체 목록 가져온 일자로 설정, 일단 YaDeals.creation_time 일자부분으로 사용
 view Object 화면 구성에 필요한 정보 
 - title Object 패키지 이름  
 - - en String 패키지 이름(영어) # '' 로 기본설정
 - - ko String 패키지 이름(한글) # YaDeal.name 값 사용
 - description Object 패키지 설명에 필요한 정보
 - - images Array-String 패키지 이미지 # YaDeal.images 사용
 - - socialImsages Array-String 상세 설명 이미지 신규 필드
 - - pictograms Object 픽토그램 정보 (closed_day, valid_period, usage_guideline, parking_info, need_reservation, support_number, registration_period, use_today) 신규 필드
 - - - {pictogram-key} String 픽토그램 내용 (위 8개중 한개 있는 경우에만 키로 처리) 신규 필드
 - - contents Object 패키지 상세 내용
 - - - en String 패키지 상세 내용(영어) # null 로 기본설정
 - - - ko String 패키지 상세 내용(한글) # YaDeal.deal_introduce 값 사용 없는 경우 deal_info, 이것도 없으면 name을 사용
 - - warnings Object 패키지 주의사항
 - - - en String 패키지 주의사항(영어) # null 로 기본설정
 - - - ko String 패키지 주의사항(한글) # YaDeal.notice_info 값
 - - highlights Object 패키지의 강조되는 설명
 - - - en String 패키지의 강조되는 설명 (영어) # null 로 설정
 - - - ko String 패키지의 강조되는 (한글)  # YaDeal.deal_point_desc
 - - additionalInfo Object 패키지 추가적인 설명
 - - - en String 패키지 추가적인 정보 (영어) # null 로 기본설정
 - - - ko String 패키지 추가적인 정보 (한글) # null 로 기본설정
 - - additional_more_information Object 패키지 좀더 추가되는 정보
 - - - en String 패키지 좀더 추가되는 정보 (영어) # null 로 기본설정
 - - - ko String 패키지 좀더 추가되는 정보 (한글) # null 로 기본설정
 - - includes Object 패키지에 포함되는 정보
 - - - en String 패키지에 포함되는 정보 (영어) # null 로 기본설정
 - - - ko String 패키지에 포함되는 정보 (한글) # null 로 기본설정
 - - exclude Object 패키지에 포함되지 않는 정보
 - - - en String 패키지에 포함되지 않는 정보 (영어) # null 로 기본설정
 - - - ko String 패키지에 포함되지 않는 정보 (한글) # null 로 기본설정
 - - businessHour Object 운영 시간 정보
 - - - en String 운영 시간 정보 (영어) # null 로 기본설정
 - - - ko String 운영 시간 정보 (한글) # YaDeal.store_info 
 - - refundPolicy Object 환불 규정
 - - - en String 환불 규정 (영어) # null 로 기본설정
 - - - ko String 환불 규정 (한글) # YaDeal.refund_info
 - - meetingPolicy Object 접선 규정
 - - - en String 접선 규정 (영어) # null 로 기본설정
 - - - ko String 접선 규정 (한글) # YaDeal.usage_info
 - - voucher Object Voucher 정보
 - - - type Array-String Voucher 타입 e-voucher : 이메일 voucher m-voucher : 모바일 voucher
 - - - useDescription Object 사용 설명
 - - - - en String 사용 설명 (영어) # null 로 기본설정
 - - - - ko String 사용 설명 (한글) # null 로 기본설정
 - - itineraries Object 경유지에 대한 정보
 - - - en String 경유지에 대한 정보 (영어) # null 로 기본설정
 - - - ko String 경유지에 대한 정보 (한글) # null 로 기본설정
 packageId String 패키지 고유 번호  # YNJ + YaDeal.id 
 updateAt String 상품 데이터 수정 날짜 # YaDeals.creation_time
 properties Object 처리에 필요한 데이터
 - map Array-Object 지도 정보 
 - - pos Object 지도 위도, 경도 정보 
 - - - latitude String 위도 정보 # YaDeal.latitude as String
 - - - longitude String 경도 정보 # YaDeal.longitude as String
 - - city Object 도시 정보 
 - - - code String 도시 코드 # YaDeal.locations - depth가 1인 값 없으면 location 1번째 값으로 GDF 값으로 매핑 처리
 - - - name Object 도시 이름 
 - - - - en String 도시 이름(영어) # '' 로 기본설정
 - - - - ko String 도시 이름(한글) # YaDeal.locations - depth가 1인 값 없으면 location 1번째 값.
 - - country Object 나라 정보 
 - - - code String 나라 코드 # 'KR'
 - - - name Object 나라 이름 
 - - - - en String 나라 이름(영어) # null로 처리
 - - - - ko String 나라 이름(한글) # '대한민국'
 - limits Object 제한 사항 정보 
 - - age Object 나이 정보 
 - - - min Number 최소 나이 정보 나이 정보를 받지 못했을 경우 값 : -1 # -1로 기본 설정 
 - - pax Object 수용 인원 정보 
 - - - max Number 최대 수용 인원 정보 # 9999로 기본 설정
 - - - min Number 최소 수용 인원 정보 # 1로 기본 설정
 - - valid Object 유효 기간 정보 
 - - - from String 유효 기간 시작 정보 # YaDeal.sales_period.start_date
 - - - to String 유효 기간 끝 정보  # YaDeal.sales_period.end_date
 - supply Object 제공 정보 
 - - price Object 가격 정보 
 - - - min Number 최소 가격 정보 상품 가격 정보를 받지 못했을 경우 값 : -1 #  YaDeal.gajago_sale_price
 - - - currency String 환율 정보 # 'krw' 원화 기본
 - - policy Object 정책 정보 
 - - - airportPickup Boolean 공항 픽업 여부 # false로 기본 설정
 - - - wheelChair Boolean 휠체어 가능 여부 # false로 기본 설정
 - - - skipLine Boolean 줄 대기 여부 # false로 기본 설정
 - - - hotelPickup Boolean 호텔 픽업 여부 # false로 기본 설정
 - - - freeSaleOnly Boolean FreeSale 여부 # false로 기본 설정
 - - - mobileVoucher Boolean MobileVoucher 여부 # false로 기본 설정
 - - - businessHour Boolean BusinessHour 여부 #  false로 기본 설정
 - - - isRefundable Boolean 환불 가능 여부 # True로 설정 # TODO 환불불가여부를 상정한 처리이므로 아니라면 수정 필요
 - - - instantConfirmation Boolean 예약 즉시 확정 여부 # False로 처리
 - - - hasMeetingInfo Boolean 접선 정보 여부 # usageInfo값을 meetingInfo에 넣었다면 true
 - - - firstAvailabilityDate Date 처음 예약 가능한 날짜 ex) ‘2019-12-20’ or null # 일단 null 로 처리하나 period값을 검색하면 처리는 가능하다 
 - - guideLanguages Array-String 가이드 언어 # 실제 데이터상 단수라서 단수로 처리. null로 설정
 - - writtenLanguages Array-String 언어 정보 # null로 설정
 - category Array-Object 카테고리 정보 
 - - supplier String 공급사 카테고리 정보 # Activities & Experiences 로 일단 처리
 - - code String 카테고리 코드 # YaDeal.type[1] 값으로 처리. 2 depth값의 GDF 코드 매핑 정보
 - - name Object 카테고리 이름 
 - - - en String 카테고리 이름(영어) # null 로 처리
 - - - ko String 카테고리 이름(한글) # GDF 코드값의 표시 텍스트 정보로 처리
